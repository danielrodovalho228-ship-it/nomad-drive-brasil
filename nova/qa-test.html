QA Test Console — Fluxo de Reserva

Console de testes end-to-end pra equipe de QA. Cria reserva real → paga via Stripe TEST → recebe código do cofre → desbloqueia → devolve → avalia.

📋 Antes de começar (1 minuto)

  1. Use seu e-mail real (ele recebe os e-mails do fluxo).
  2. No checkout Stripe, use o cartão de teste: 4242 4242 4242 4242 · validade qualquer futura · CVC qualquer 3 dígitos · CEP qualquer.
  3. O fluxo completo leva ~3 minutos. Você pode pular passos pra testar cenários isolados.
