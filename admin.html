Configurações

Este painel é uma demonstração visual da operação. O acesso real exigirá autenticação da equipe (login/super admin). Módulos como cadastro de usuários, integrações e permissões serão adicionados conforme a plataforma evolui.

Nenhum dado sensível é publicado nesta página.

🔧 Stripe Connect — Configurar payouts manuais

Proprietários (Connected Accounts Express) vêm com schedule "daily" por default. Pra ativar saques parciais (Sprint 2 Fase 32) precisam estar em modo manual. O Dashboard do Stripe não permite editar Express via UI — só via API.
